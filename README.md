## 🏥 About The Project

**Kovai Lakshmi Wellness Centre** is a modern, responsive website built to promote traditional healing therapies and pain relief treatments in Coimbatore.

The website focuses on:

* 🌿 Natural pain relief treatments
* 🧘 Traditional therapies like Varma & Acupuncture
* 🥗 Nutrition consultation
* 📍 Local SEO for better visibility in Google

---

## 🚀 Features

* ✅ Fully responsive (Mobile + Desktop)
* ✅ Tamil + English bilingual content
* ✅ Clean & modern UI design
* ✅ SEO optimized structure
* ✅ WhatsApp integration

* ✅ Google Maps integration
* ✅ Fast loading & performance optimized
